Line search

Description

Available line search algorithms

  • hz_linesearch! , the default line search algorithm
  • backtracking_linesearch!
  • interpolating_linesearch!
  • mt_linesearch!

The default line search algorithm is taken from the Conjugate Gradient implementation by Hager and Zhang (HZ).
